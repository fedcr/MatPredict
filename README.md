# MatPredict
**MatPredict** is an AI-powered Polymer Alloy Calculator for 3D printing makers. 
An open-source, free tool that helps you calculate the exact mixture of plastic waste, virgin pellets, and masterbatches to extrude custom, high-quality filament.

## Features
- **AI Formulation Engine:** Calculate recipe percentages in seconds using Gemini API.
- **Manual Configurator:** Full control over texture, odor, and mechanical parameters.
- **Laboratory:** Log experimental data and contribute to the global database.
- **Open Data:** Export your recipes and data as CSV.

## How to use
1. Open the website: https://fedcr.github.io/MatPredict/
2. (Optional) Save your own Gemini API Key in the settings for AI-powered suggestions.
3. Configure your target mass and material goals.
4. Calculate and start extruding!
